**Шаг 1:**\
\
Добавьте бота @ai2media\_grabber в ваш канал, дайте права и добавьте канал в список.\
(скопруйте адрес канала, вставьте в поле ТГ name и нажмите сохранить)

3\) Выберите “Темы/категории поиска” из списка\
